# 🚀 Timeline PM: 성공을 설계하는 도구 (Vision & Roadmap)

본 문서는 사용자의 비전과 향후 개발 방향을 기록한 공식 로드맵입니다. 단순한 일정 관리를 넘어, 전문가로 성장하기 위한 **'몰입 시간 관리(순공 시간)'**와 **'다중 프로젝트 통합 관리'**를 목표로 합니다.

---

## 📅 Phase 1: 순수 몰입 시간 관리 (Stopwatch & Deep Work)
성공한 사람들의 습관인 '1만 시간의 법칙'과 '순수 투자 시간'을 관리하는 핵심 기능입니다.

- **작업별 목표 시간 설정**: 각 작업 카드에 "일일 목표 투자 시간" 속성 추가.
- **실시간 스톱워치**: 
  - 특정 작업 선택 후 '시작' 버튼 클릭 시 스톱워치 가동.
  - 종료 시 해당 날짜의 작업 세션으로 저장.
- **성공 메트릭스**: 오늘 목표 대비 실제 집중 시간(Percentage) 시각화.

## 📂 Phase 2: 다중 프로젝트 시스템 (Multi-Project Engine)
하나의 앱에서 여러 독립된 프로젝트를 관리하는 구조로 확장합니다.

- **프로젝트 대시보드 (Home)**: 프로젝트 목록을 카드 형태로 한눈에 보는 메인 화면.
- **카테고리 분류**: IT, 건설, 개인 학습 등 도메인별 프로젝트 그룹화.
- **데이터 격리**: 프로젝트별로 독립된 Task 목록과 기록 보관.

## 📊 Phase 3: 인사이트 분석 대시보드 (Insights & Analytics)
기록된 데이터를 기반으로 자기 객관화가 가능한 통계를 제공합니다.

- **영혼의 투자 차트**: 내가 어떤 프로젝트/작업에 가장 많은 시간을 쏟았는지 분석.
- **성공 패턴 달력**: 목표 시간을 달성한 날들을 시각적으로 보상(하이라이트).
- **병목 구간 추적**: 계획 대비 실제 진행이 더딘 구간을 지능적으로 탐지.

## 🌐 Phase 4: 배포 및 피드백 (Deployment & Feedback)
실제 사용자(테스터)들이 브라우저에서 사용할 수 있도록 세상에 내놓습니다.

- **Static Hosting (GitHub/Firebase)**: 서버 설정 없이 로컬 스토리지를 활용한 즉시 배포.
- **피드백 루프**: 시험 사용자들로부터 개선 사항을 수집하기 위한 채널 마련.

> 본 툴은 단순히 '할 일'을 적는 곳이 아니라, 사용자의 '성장의 증거'를 기록하는 공간입니다.
